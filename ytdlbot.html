YouTubeDLBot

Downloads videos, music, and much more.

The source code for the bot has been lost. Until the bot is rewritten, RIP YouTubeDLBot.

Commands

The bot's prefix is "yt!"

  • help - Shows a list of commands.
  • update - Updates the version of yt-dlp that the bot uses. The bot will run this by itself sometimes.
  • dl - The main event. This command acts exactly like yt-dlp; same arguments, same format, etc.
  • changelog - Shows recent changes to the bot

Further notes

The bot constantly restarts, even during command execution. If your command freezes, rerun it.
